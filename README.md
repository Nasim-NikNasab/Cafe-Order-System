# Cafe Order System

## About the Project

This project is a simple **Cafe Order System** developed with **Python**. It allows customers to place coffee orders, calculates the total price, applies a discount when needed, and stores the order history.

## Features

* Customer name input
* Coffee order processing
* Total price calculation
* Automatic discount calculation
* Order validation based on available cups
* Order history display

## Technologies Used

* Python

## How to Run

1. Install Python.
2. Download or clone the project.
3. Run the Python file.
4. Enter the customer name and the number of coffee cups.
5. The program calculates the total price and displays the order history.
